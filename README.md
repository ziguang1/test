# 贪吃蛇游戏项目

这是一个使用纯JavaScript、HTML和CSS实现的贪吃蛇游戏。

## 游戏功能

- 🎮 完整的贪吃蛇游戏逻辑
- 🎨 现代化的UI设计
- 📱 响应式布局，支持移动设备
- 🎯 键盘控制（方向键或WASD）
- 📱 触摸控制（移动设备）
- ⏸️ 暂停/继续功能
- 🏆 最高分记录（使用localStorage）
- 🎮 游戏结束界面

## 如何运行

1. 确保你安装了Python 3
2. 在项目目录中运行以下命令：
   ```bash
   python3 -m http.server 8000
   ```
3. 打开浏览器访问：http://localhost:8000/snake-game.html

## 游戏控制

- **方向键** 或 **WASD键**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **触摸屏**：滑动控制（移动设备）

## 游戏规则

1. 使用方向键控制蛇的移动
2. 吃到红色食物可以增加分数和蛇的长度
3. 撞到墙壁或自己的身体会导致游戏结束
4. 目标是获得尽可能高的分数

## 技术栈

- **HTML5**: 游戏结构
- **CSS3**: 样式和动画
- **JavaScript (ES6+)**: 游戏逻辑
- **Canvas API**: 游戏渲染
- **localStorage**: 最高分存储

## 项目特点

1. **面向对象设计**: 使用ES6类封装游戏逻辑
2. **模块化代码**: 清晰的代码结构和职责分离
3. **响应式设计**: 适配不同屏幕尺寸
4. **性能优化**: 高效的渲染和游戏循环
5. **用户体验**: 直观的界面和流畅的控制
